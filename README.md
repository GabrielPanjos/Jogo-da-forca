# Jogo da Forca em Python

Este é um simples jogo da forca implementado em Python. O objetivo do jogo é adivinhar uma palavra sorteada, caracter por caracter, dentro de um número limitado de tentativas.

## Funcionalidades

- O jogo escolhe aleatoriamente uma palavra de categorias como: **animais**, **comidas** e **desenhos**.
- O jogador adivinha letras uma por uma. Se a letra for correta, ela é revelada na palavra.
- O jogo continua até que o jogador acerte todas as letras da palavra ou desista.
